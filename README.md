# Личный кабинет студента

Веб-приложение для управления академической информацией студентов.

## Основные цели
- Предоставить студентам удобный доступ к персональным данным и учебным материалам.
- Автоматизировать процесс взаимодействия "Студент-Преподаватель".
- Обеспечить просмотр расписания, оценок и сдачу заданий.

## Базовый функционал (план)
- **Для студента:**
    - Просмотр профиля.
    - Просмотр расписания занятий.
    - Просмотр текущих оценок.
    - Загрузка домашних заданий.
- **Для преподавателя:**
    - Просмотр списка студентов.
    - Выставление оценок.
    - Публикация заданий и материалов.
